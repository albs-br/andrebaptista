Shoot'em up vertical scrollng game for MSX

Some screenshots of the game running on an emulator.

While digging old books, magazines and current MSX community resources, I had an idea for a game. As it's on the github description: Shoot'em up vertical scrolling game for MSX 1 made in Assembly, inspired by 8-bit classics such as 1942.

At same time I stumbled with the MSX Dev, a contest of homebrew games for MSX machines, which since 2003 has revealed more than 150 new games for the platform. And the submission period were open. So, why not?

As I have no physical machine to test, all the work was done with emulators, such as openMSX, Emulicious and Web MSX. Only on later development stages, some people of the MSX community tested the game on real hardware.

My setup on this work:

  • VS Code with many cool extensions;
  • tniASM 0.45;
  • MSX Pen to test some small pieces of code on isolation

The game can be played online on Filehunter, source code on github, MSX Dev entry here, related threads on MSX.org here and here.
