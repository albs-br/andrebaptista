JACA stands for Just Another Homebrew CPU Again. It's an 8-bit computer projected and made from scratch just from regular TTL ICs.

Part of the computer built on breadboards

I've always had the idea of joining my (meager) electronics knowledge with the programming knowledge. To know where is the lost link between the instruction in Assembly language and the logic gates that brings life to it.

The project progress is documented on the Hackaday site.

I started by doing the logical project on a software called Logisim. First I've done a small 4 bit CPU as a POC (Proof Of Concept), and then the 8 bit version:

8 bit CPU done on Logisim

Video of the computer simulated on Logisim running some test programs

I also made an Assembler program in C#/Windows Forms. An assembler is a program that converts the Assembly language (a mnemonic that is easier for a human to read) in the Machine Code (the zeroes and ones that is effectivelly the only "language" the computer understands). Here is it:

Screenshot of the Assembler program

That was the easy part. So I started building the computer on breadboards, module by module. The first one (the simplest) was very straightfoward, but as I added more and more modules the problems started to arise. It costed me MONTHS of work until I realized that with these very poor quality chinese breadboards I wouldn't go nowhere.

Video of a partial assembly of the computer

Then I decided to move on to another approach and rebuilt the computer on Printed Circuit Boards (PCBs), designing the circuits on the EasyEDA online platform and ordering the PCBs with the JLC PCB.
