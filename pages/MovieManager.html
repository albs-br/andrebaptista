Movie Manager is a small web app to track the movies I watched.

Screenshot of the app running on a large screen.

This is a POC (proof of concept) I have done to learn the MEAN stack. Originally its RESTful API was made in Microsoft's Web Api platform, now is in NodeJS. The client-side is still on Azure (link below).

The application's goal is to keep a list of movies the user watched/wants to watch. It should be usable on mobile with the look and feel of a native app.

On the back-end:

  • ExpressJS running over NodeJS, using Mongoose to keep things easy, hosted on the great Glitch
  • MongoDB hosted on Mlab.com
On the front-end:
  • AngularJS 1.5, for a while it will be hosted in Microsoft's Azure
  • It uses EcmaScript 6, so a recent version of the browser is required
  • Chrome use is recommended
  • url: moviemanager.azurewebsites.net
