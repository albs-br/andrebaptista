NESPi is just a Raspberry Pi 2 enclosed in a Classic NES styled case, running Retropie. I'm not even sure if this should be called a "project", but here it is.

NESPi with two controllers: a cheap USB wired ressembling the original two buttons NES joypad and a wireless multi-button with analog sticks, both from Data Frog.

The best tutorial I found to make this project was this one.

All components are easily found on internet, at Ebay for example. Special care should be taken with the power source as many cheap Ebay phone chargers don't deliver the current they promise. When is underpowered a lightning symbol appears on the upper right corner of the screen and unpredictable behaviour is possible (usually reboot or frozen).

Update:

Recently the NESPi maker, Retroflag, released a SNES styled case for Raspberry. It's the same quality standard, with some improvements, such as soft shutdown/reset.

Also, the USB controller included on the Deluxe version is, according to some user reviews on Amazon, the best controller for Retropie. Even better than the expansive 8BitDO wireless gamepadas. Very well built, almost zero lag (is wired), the digital pad is "tight", unlike the cheap Ebay clones, which when you press down arrow for example it unadvertidally presses left or right.
